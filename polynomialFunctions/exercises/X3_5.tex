Which of these could be the graph of the function $\displaystyle p(x) = -x^3+3x^2-4$? 
\begin{multipleChoice}
	\choice[correct]{Graph (A)}
	\choice{Graph (B)}
	\choice{Graph (C)}
	\choice{Graph (D)}
\end{multipleChoice}	
\begin{feedback}
	Notice that all these graphs have the same zeroes.  Use that to find the zeroes and their multiplicities for $p$.
