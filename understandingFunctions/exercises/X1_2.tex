%\outcome{Inverse functions.}


The function $g$ given by the formula
\[ g(x) = \frac{x+1}{3x-2} \]
is a one-to-one function, so it has an inverse.  Find a formula for the inverse function, $g^{-1}(x)$.

\[ g^{-1}(x) = \answer{\frac{2x+1}{3x-1}}\]
